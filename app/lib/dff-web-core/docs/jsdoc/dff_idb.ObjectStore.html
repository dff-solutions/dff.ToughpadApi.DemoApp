Class: ObjectStore

dff/idb.ObjectStore

new ObjectStore(openDBPromise, name, access)

Abstracts access to browsers object store API which allows access to indexedDB object store. Provides a general error handling and supports handling multiple documents at once.
Parameters:
Name Type Description
openDBPromise Promise Promise resolving when idb is opened successfull.
name string Name of the object store.
access string readwrite/readonly access attribute to object store.
